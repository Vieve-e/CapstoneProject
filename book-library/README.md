# Book Library

## Description
The Book Library project is designed to help users organize and access their eBooks easily. It promotes a reading culture by offering features to search, find, and categorize books.

## Features
- Add, edit, and delete books in the library.
- Search for books by title, author, or genre.
- Save books locally for offline access.

## Technologies Used
- React
- Tailwind CSS
- Node.js (optional for backend)
